All blend files. To export them to tiles:
	1. Open project into a blender
	2. Install super duper batch exporter extension https://extensions.blender.org/add-ons/superduperbatchexporter
	3. Select folder where you should export individual files
	4. Select gltf format with all rendered object visible
	REMEBER TO SAVE A COLISION OBJECTS SHOULD HAVE A 4 digit index number "0001" and  "-COL" suffix ex.  0001-Object-col
